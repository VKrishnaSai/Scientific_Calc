# Scientific Calculator Project

This project contains implementations of a scientific calculator in multiple languages.

## Directory Structure

- `main/`: Maven-based Java project.
  - `pom.xml`: Maven configuration file.
  - `src/main/java/calc/App.java`: Java implementation of the scientific calculator, combining classes and main logic.
  - `src/test/java/calc/AppTest.java`: JUnit tests for the calculator functionality.
- `target/`: Compiled Java classes (generated by Maven).

## Running the Java Version

Navigate to the `main` directory and use Maven:

```bash
cd main
mvn compile
mvn exec:java -Dexec.mainClass="calc.App"
```

## Running Tests

For Java tests:

```bash
cd main
mvn test
```

## Features

- Arithmetic operations: addition, subtraction, multiplication, division
- Power and root operations: exponentiation, square root, cube root
